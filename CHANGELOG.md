# Changelog / 更新日志

All notable changes to this project will be documented in this file.

## [v0.4.1] - 2026-04-22

### ✨ 新功能
- 新增风格资产数据模型、资产中心管理能力与项目级风格选择
- 支持在角色、场景创建与生成链路中直接复用风格资产
- 为角色、场景、分镜、变体和视频任务挂接风格快照，统一风格解析来源

### 🐛 修复
- 修复风格资产流程中的鉴权与会话稳定性问题
- 修复项目默认风格清空与选择链路中的交互问题
- 修复风格解析在 worker 和项目过滤场景下的兼容问题

### ⚙️ 优化
- 刷新 Director Studio 工作台与资产中心界面
- 补充风格资产相关的接口契约、集成测试与单元测试覆盖

## [v0.2] - 2026-02-28

### ✨ 新功能
- 增加 OpenAI 兼容图片、视频格式支持

### 🐛 修复
- 修复默认模型配置后项目模型需要二次选择的问题
- 修复部分情况 resolution 无法读取的问题
- 修复模型链路为 LangGraph
- 修复默认参数无选择问题
- 修复关闭计费依然触发计费问题
- 修复 openai-compatible 被误判为原生 OpenAI 推理问题
- 修复 JSON 解析失败问题

### ⚙️ 优化
- 修改为默认计费 off
- 增强提示词 JSON 格式限制
